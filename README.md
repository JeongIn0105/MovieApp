# 🎬 영화 예매 앱 팀 프로젝트

> 영화 예매 앱의 UI와 기능을 설계하고 구현하고, 개발자로서 사용자 경험(UX)를 기술적으로 풀어내는 과정 경험.


## 📋 프로젝트 개요

 사용자가 원하는 영화를 빠르고 정확하게 예매하고, 예매 내역을 안전하게 관리할 수 있도록 기술과 UX가 완벽하게 조화를 이루는 구조를 만드는 것입니다.
 전반적으로 영화 예매 앱으로, 전반적으로 영화 정보를 조회하고 검색하고, 사용자가 원하는 영화를 예매할 수 있는 팀 프로젝트입니다.

## 📋 프로젝트 설명
1. **`UIKit`을 활용한 화면 구성 및 화면 전환**
    - 화면구성: `UILabel`, `UITextField`, `UIButton`, `UICollectionView`, `UITableView` 등을 활용하여 화면을 구성하였다.
    - 화면전환: `NavigationController`와 `TabBarController`를 사용하여 로그인 → 영화 목록 → 영화 상세 → 마이페이지 등의 화면 전환을 구현하였다.
    
2. **TMDB API를 활용한 영화 데이터 조회**
    
    - `URLSession`을 사용하여 TMDB API와 네트워크 통신을 구현하였다.
    
    - 무비 차트, 현재 상영작, 상영 예정 영화 데이터를 가져와 화면에 표시하였다.
    
    - 영화 검색 기능을 통해 사용자가 원하는 영화를 조회할 수 있도록 구현하였다.
  
    - TMDB API : https://developer.themoviedb.org/docs/getting-started

3. **비동기 처리 및 검색 기능 구현**
    
    - `DispatchGroup`을 사용하여 여러 영화 API 요청을 동시에 처리하였다.
  
    - `Debounce` 방식을 적용하여 검색 시 불필요한 API 호출을 줄이고 성능을 개선하였다.
    
4. **UserDefaults를 활용한 사용자 정보 관리**
    
    - 회원가입 시 사용자 정보를 `UserDefaults`에 저장하였다.
    
    - 로그인 시 저장된 정보를 비교하여 인증을 수행하였다.
  
    - 마이 페이지에서 사용자 정보를 불러와 표시하도록 구현하였다.

## 📱 각 주요 기능
1. **로그인 및 회원가입 화면**
   - 사용자 아이디와 비밀번호를 입력하여 로그인 기능을 구현하였다.
   - 회원가입 화면에서 사용자 정보를 입력하면 UserDefaults에 저장된다.
   - 로그인 성공 시 `TabBarController` 화면으로 이동하도록 구현하였다.
  
  
2. **영화 목록 페이지**
   - TMDB API를 이용하여 영화 데이터를 불러온다.
   - 인기 영화, 현재 상영작, 상영 예정 영화 목록을 표시한다.
   - `UICollectionView`를 사용하여 영화 포스터를 가로 스크롤 형태로 구현하였다.
   - 영화 포스터를 선택하면 영화 상세 페이지로 이동한다.
  
3. **영화 세부 페이지**
   - 선택한 영화의 상세 정보를 표시한다.
   - 영화 포스터, 줄거리, 평점 등의 정보를 보여준다.
   - 영화 통계 정보(예매율, 누적관객수, 에그지수)를 화면에 표시한다.

4. **영화 검색 페이지**
   - 검색창에 영화 제목을 입력하여 영화를 검색할 수 있다.
   - TMDB API 검색 기능을 활용하여 검색 결과를 표시한다.
   - `Debounce` 기능을 적용하여 불필요한 API 요청을 줄였다.
   - 무한 스크롤을 적용하여 추가 영화 데이터를 불러올 수 있도록 구현하였다.

5. **영화 예매 입력 페이지**
   - 사용자가 영화 예매 정보를 입력할 수 있는 화면이다.
   - 예매 영화, 좌석, 날짜 및 시간 등의 정보를 입력한다.
   - 입력된 예매 정보는 마이페이지에서 확인할 수 있도록 구현하였다.
     
6. **마이 페이지**
   - 로그인한 사용자의 정보를 확인할 수 있는 화면이다.
   - `UserDefaults`에 저장된 이름, 아이디, 이메일 정보를 표시한다.
   - `PHPickerViewController`를 사용하여 프로필 이미지를 변경할 수 있다.
   - 사용자의 영화 예매 및 결제 내역을 확인할 수 있도록 구현하였다.
7. **도전 기능**
   - 로그아웃 기능
   - 로그인 아이디 비밀번호 나갔다 들어와도 써져있는 자동완성기능
   - 아이디 찾기, 비밀번호 찾기
   - 프로필 사진 올리기
   - 내 정보에서 이름 누르고 수정할 수 있는 기능
